GNR638: Machine Learning for Remote Sensing II

Instructor: Prof. Biplab Banerjee

Type: Graduate Elective Course

Prerequisites: A First course in Machine Learning

When: Tue./Fri., 19:00-20:30 in LH 102

Assistants: Saurabh Kumar

Moodle link: GNR638.

Overview

This is an advanced level course on machine learning motivated via applications to Remote Sensing and Satellite data analysis. In this course, students will learn about various state of the art machine learning techniques from recent research papers. The students are expected to have take a first course in machine learning to better understand the content.

Content and Syllabus

Review of machine learning fundamentals, Introduction to deep learning, Convolutional networks.
